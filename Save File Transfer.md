> 👉 **This guide explains how to transfer your GnollHack save files between different devices and platforms using the Save Transfer feature in the Vault.**

## ⚙️ Prerequisites

Before you can transfer save files, ensure you meet the following requirements:

- **Internet Connection:** An active internet connection is required on both the source and target devices.
- **GnollHack Account:** A registered GnollHack server account is required. You can sign up or log in by navigating to [[Settings]] and opening the **Server Posting** section.
- **Compatible Versions:** Both devices must run compatible versions of the game (modern port version 3.1 or newer). For details, check [[Save Game Compatibility]].

## 📤 Uploading a Save File

To transfer a save file off your current device:

1. Open the game and go to the main menu.
2. Tap the **Vault** button.
3. Tap **Save Transfer**.
4. Set the selection picker at the bottom to **Upload**.
5. Select the character save file you want to transfer from the list.
6. Tap the **Upload** button.

> ℹ️ **Note:** A successful upload moves the save file to the cloud and automatically deletes the local copy from your current device to prevent duplication.

## 📥 Downloading a Save File

To download a previously uploaded save file onto a new device:

1. Ensure you are logged into the same GnollHack server account in [[settings]].
2. Tap the **Vault** button on the main menu.
3. Tap **Save Transfer**.
4. Set the selection picker at the bottom to **Download**.
5. Select the character save file you wish to download from the list of cloud saves.
6. Tap the **Download** button.

> ℹ️ **Note:** Once the download is complete, the save file is removed from the cloud, and you can resume playing the character locally on this device. If a local save with the same name already exists, you will be prompted to overwrite it.

## 💡 Important Details

- **Move Semantics:** Save Transfer acts as a *move* operation, not a copy. A save file can only exist on one device or in the cloud at any given time.
- **Save File Tracking:** On Windows, you must have **Save File Tracking** enabled in your settings before you can use Server Posting and transfer saves. Check [[Save File Tracking]] for details.
- **Corrupted or Invalid Saves:** If a save file becomes corrupted or invalid, it will display a **Delete** button instead of a Transfer option, allowing you to manually clean it up.
