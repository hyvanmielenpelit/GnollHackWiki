This is a curated list of user reviews of the **Android version of GnollHack** on the Google Play Store.

### ⭐⭐⭐⭐⭐ (9 June, 2025)

> I don't have the words... this reboot of an old classic is flawless. It's the best version of the old Moria from the 80s I've ever played, with tons of added goodies and mechanics. I only wish there was more! More levels and NPCs! More artifacts and items! But that's straight greed. This Nethack spinoff is a straight up masterpiece. 11/10, best mobile game of all time. 🔥🙌

### ⭐⭐⭐⭐⭐ (22 December, 2023)

> Amazing adaptation of the OG granddaddy of roguelikes, the classic Nethack. But with an actual good control scheme for mobile and nice visuals and sound. Hands down the best "port" of nethack I've seen. Better than even Vulture on PC & gurr's mobile port. Accessibility features baked in like the npc at the entrance and streamlined HUD make it way more approachable for new players. A real upgrade for a game notorious for its complexity. Amazing work 10/10. I might actually finish a run now

### ⭐⭐⭐⭐⭐ (13 February, 2024)

> I never was able to get into traditional roguelikes because of how arcane the controls can be, and how much meta-knowledge you're expected to learn with next to no QoL. Enter GnollHack -​ the team(?) behind this has built a fantastic UI that helps IMMENSELY with the learning curve of Nethack. It's amazing that GnollHack manages to make such a classic system so much more approachable, and without sacrificing any of the late-game challenge that veterans look for. Also, there's a full wiki! 10/10

### ⭐⭐⭐⭐⭐ (23 August, 2025)
