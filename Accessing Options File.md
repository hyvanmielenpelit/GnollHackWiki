The method for accessing and modifying your game options depends entirely on which platform or version of GnollHack you are playing. 

| Platform / Version | How to Access | File Location / Name |
| :--- | :--- | :--- |
| 📱 **Modern Ports**<br>(Android, iOS, and Windows) | You need to enable **Developer Mode** in the settings. Then **Options** will appear in the main menu. | *Managed via in-game menu* |
| 🐧 **Linux** | Open and edit the text file directly using any text editor. | `~/.gnollhackrc`<br>(Located in your home folder) |
| 🌐 **Linux Server**<br>(Hardfought, server.gnollhack.com) | You use the server's web interface to edit the options file. | *Managed via Server UI* |
| 🪟 **Windows Legacy Ports** | Open and edit the text file directly using any text editor. | `defaults.gnh`<br>(Located in the same folder as the executable) |
