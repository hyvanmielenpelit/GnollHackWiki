Save game compatibility depends on the **library version** of GnollHack included in various port builds. The following builds and port versions are compatible with each other.

| Compatibility Group | GnollHack Library Version Number | Modern Port Version Number | Cross-platform Compatibility<sup>1</sup> | Save Game Transfer Behavior |
|:-------------------:|:--------------------------------:|:--------------------------:|:---------------------|:-----|
| C | 4.3.0 Build 1 – Current | 3.0 – Current | All platforms, guaranteed | **Official support for save game transfer.** Keeps transferred save game files eligible for top scores after transfer.  |
| B | 4.2.0 Build 1 – 4.2.0 Build 146 | 1.73 – 2.146 | All platforms, not guaranteed | Moving save games across platforms makes them non-scoring. |
| A | 4.1.2 Build 7 – 4.1.3 Build 55 | 1.13 – 1.72 | Android & iOS | Save game transfer not possible |

- <sup>1</sup> **Cross-platform compatibility** means if you can transfer saved games from one platform to another.
